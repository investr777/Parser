====================================================
НАЗВАНИЕ ПРОЕКТА
====================================================
Parser
----------------------------------------------------


====================================================
ФУНКЦИОНАЛ
====================================================
Пользователь вводит URL адрес, после чего нажимает на кнопку "Анализировать", происходит анализ URL адреса и выводится информация в виде:
колечества найденных ссылок, их мена и URL адреса.
По нажатию на URL адрес в таблице, он автоматически подставляется в поле ввода.
----------------------------------------------------


====================================================
СБОРКА ПРОЕКТА
====================================================
Сборку проекта в jar осуществляется командой: gradle build
----------------------------------------------------


====================================================
ЗАПУСК ПРОЕКТА
====================================================
Запуск проекта осуществляется командой: gradle runApp
В браузере перейти на страницу http://localhost:8080/
